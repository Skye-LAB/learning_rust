# Data Types

## Tipe Scalar

Tipe scalar mewakili nilai tunggal, diantaranya integer, floating-point, numbers, Boolean dan karakter.

### Integer

Di Rust integer dibagi menjadi dua, yaitu signed(diawali dengan `i`) dan unsigned(diawali dengan `u`). berikut tabel daftar integer:

| Length | Signed | Unsigned |
| :----: | :----: | :------: |
| 8-bit  | i8     | u8       |
| 16-bit | i16    | u16      |
| 32-bit | i32    | u32      |
| 64-bit | i64    | u64      |
| 128-bit| i128   | u128     |
| arch   | isize  | usize    |
